> CORE FEATURES

Everything you need to manage your audio production ecosystem.

⚡ Plugin Scanner

Scans VST2, VST3, and AU plugins from system directories. Shows architecture badges (ARM64, x86_64, Universal). Streams results in real-time.

🎵 Audio Sample Scanner

Discovers WAV, FLAC, AIFF, MP3, OGG samples with metadata. Click to play. Floating player with volume, speed, loop, and visualizer.

🎧 DAW Project Scanner

Finds projects across 14+ DAW formats. Double-click to open directly in your DAW. Ableton, Logic, FL Studio, REAPER, and more.

🎴 Preset Scanner

Catalogs plugin presets across all formats. Search, filter, and reveal in Finder with a double-click.

🔍 fzf Fuzzy Search

All search bars use fzf-style scoring with match highlighting. Supports exact ('term), prefix (^term), suffix (term$), and negation (!term).

★ Favorites

Right-click any item to add to favorites. Dedicated Favorites tab with type filtering and search. Persisted across sessions.

🔎 KVR Version Checker

Checks KVR Audio for the latest version of each plugin. Downloads available. Results cached for instant restarts.

📋 Export Everything

Export any tab to JSON, TOML, CSV, TSV, or PDF. Import from JSON or TOML. Native file dialogs with format selection.

📅 Scan History & Diff

Stores up to 50 scan snapshots. Diff any two to see plugins added, removed, or version-changed.

🎨 Cyberpunk Themes

Multiple color schemes with CRT scanline overlay. Custom schemes with 17 configurable colors. Light mode included.

♫ Floating Player

Draggable audio player docks to any corner. Volume, speed (0.25x-2x), loop, seek bar. Expands to show 50-track history with visualizer.

🛠 Multi-Select Filters

All filter dropdowns support multiple selections. Type keywords to auto-activate filters. Drag tabs to reorder.

> VISUAL INTERFACE

The cyberpunk CRT interface in action.

> BOOT SEQUENCE

Initial state. The grid awaits your scan command. Last scan results auto-load on startup so you're ready instantly.

> SCANNING AUDIO NODES

Plugins stream into the list in real-time. Progress counter shows live status. Architecture badges (ARM64, x86_64) per plugin. Hit Stop anytime -- discovered plugins are kept.

> UPDATE MATRIX LOADED

KVR Audio check in progress. Cards update incrementally with "Update Available" or "Up to Date" badges. Yellow KVR buttons link to product pages. Green download buttons for outdated plugins.

> SCAN HISTORY // DIFF ENGINE

Every scan is timestamped and archived. Select two snapshots and the diff engine shows plugins added, removed, or version-changed between them.

> USER GUIDE

How to use AUDIO_HAXOR effectively.

1

Scan Your Plugins

Click "Scan Plugins" or use "Scan All" in the header to scan plugins, samples, DAW projects, and presets simultaneously. Results stream in real-time. Use Stop to cancel anytime without losing results.

2

Check for Updates

After scanning, click "Check Updates" to query KVR Audio for each plugin. Cards update with version badges as results arrive. Green download buttons appear for outdated plugins. Results are cached -- subsequent checks are instant.

3

Search & Filter

Type in any search box for fzf-style fuzzy matching with highlighted results. Use operators: 'exact for exact match, ^prefix for prefix, !term to exclude, term1 | term2 for OR. Toggle the .* button for regex mode. Multi-select any filter dropdown to combine types.

4

Preview Audio Samples

Click any sample row to start playback. The floating player appears with volume, speed control (0.25x-2x), loop toggle, and seek bar. Drag the player to any corner. Double-click to expand and see your recently played history with a cyberpunk visualizer.

5

Open DAW Projects

Double-click any DAW project row to open it directly in its associated DAW application. If the DAW isn't installed, you'll see an error toast. Right-click for more options including Reveal in Finder.

6

Manage Favorites

Right-click any plugin, sample, DAW project, or preset and choose "Add to Favorites". Switch to the Favorites tab to see all starred items. Filter by type, search, and manage your collection.

7

Export Your Data

Click Export on any tab to save your data. Choose from JSON, TOML, CSV, TSV, or PDF format. Import previously exported data from JSON or TOML files.

8

Right-Click Everything

Every element has a context menu. Right-click plugins for KVR links, samples for playback controls, headers for stats, search boxes to toggle modes, column headers to sort, settings to toggle, and the player for controls.

> QUICK START

Get up and running in minutes.

# Clone the repo
git clone https://github.com/MenkeTechnologies/universal-plugin-update-manager.git
cd universal-plugin-update-manager

# Install dependencies
pnpm install

# Run in development mode
pnpm tauri:dev

# Build for distribution
pnpm tauri:build

Requires Node.js, pnpm, and Rust. The Tauri CLI is pulled in as a dev dependency.
